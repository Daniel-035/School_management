# Parent & Guardian App

Flutter (Dart) mobile app for parents and legal guardians. Lets families track
their child's academic progress and stay engaged with the school.

## Features (per PRD)

- **Dashboard** — snapshot of attendance, upcoming dues, latest notices; supports
  multiple child profiles (siblings).
- **Attendance Tracker** — daily attendance status; apply for leave with reason
  and date range.
- **Academic Tracker** — daily homework, upcoming deadlines, digital report
  cards, and exam schedules.
- **Fee Portal** — fee breakdown, due dates, payment history, mock payment
  gateway, and downloadable receipts.
- **Communication** — push notifications for announcements; two-way messaging
  with assigned teachers, restricted to school-defined hours.
- **School Calendar** — holidays, sports days, parent–teacher meetings, exams.

## Architecture

Feature-first clean architecture, mirroring the `staff_app`.

```
lib/
  core/           # theme, routing, constants, utils
  data/           # models, mock services, repositories
  features/
    auth/
    shell/        # bottom-nav scaffold
    dashboard/
    attendance/
    academics/    # homework, report cards, exam schedule
    fees/
    communication/
    calendar/
    profile/
  shared/widgets/ # reusable UI building blocks
  app.dart
  main.dart
```

State management: **Riverpod**.
Routing: **go_router** (declarative, deep-link friendly).
Data: in-memory mock services behind repository interfaces — swap to the real
backend later by replacing the repository implementations.

## Getting started

```bash
flutter pub get
flutter run
```

Requires Flutter 3.27+ (uses `Color.withValues`).

## Login

Sign in with your Firebase Authentication credentials. Credentials are
generated by an administrator when your account is created and emailed to your
registered address. Users cannot set their own passwords.
